These plots break down each WAR estimate into its components. The three WAR providers all use same four components:

  • Batting: Value created as a hitter relative to average
  • Fielding: Value created as a defender, relative to the average at their position
  • Baserunning: The value of their stolen bases and more complicated baserunning events like going first-to-third on a single, relative to average
  • Replacement: This value converts from the “relative to average” baseline to “relative to replacement”. It is a measure of how difficult it would be to fill this player’s position with average production.

Most plot elements are hoverable or clickable for more info


For further technical details on how these figures are produced, please refer to the Details tab.
