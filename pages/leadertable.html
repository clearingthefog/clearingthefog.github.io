Acronyms:

  • BR, FG, BP: Baseball Reference, FanGraphs, Baseball Prospectus
  • WAR: Wins Above Replacement
  • WAA: Wins Above Average
  • Batting: Value created as a hitter relative to average
  • Fielding: Value created as a defender, relative to the average at their position
  • Baserunning: The value of their stolen bases and more complicated baserunning events like going first-to-third on a single, relative to average
  • Replacement: This value converts from a relative to average baseline to relative to replacement level, given a players position and playing time. It is a measure of how difficult it would be to fill this players position with average production.
  • StdDev: Sample standard deviation, calculated as sqrt(SS / (N - 1)). For Permutation WAR, the samples are not truly i.i.d., so statistical analysis using this value should be done with caution.
  • N: Number of permutations available, used in calculating mean and standard deviation
  • PA: Plate appearances. Differences between providers in this column can be caused by some providers updating their daily stats later than others.
